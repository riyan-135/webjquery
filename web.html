APA ITU EDUTECH ?

Edutech adalah sebuah program pendidikan di bidang teknologi, yang saat ini erfokus untuk melahirkan programmer dan FullStack Developer handal, sehingga mampu bersaing dengan dunia internasional di bidang digital dan teknologi, dengan memberikan kesempatan BEASISWA hingga 100%

Dengan sistem 100% JAMINAN KERJA, kamu akan semakin yakin bahwa Edutech akan membantu kamu berkarir dan memiliki skill di bidang programming. Terlebih lagi dengan pembelajaran secara OFFLINE (tatap muka), kamu bisa lebih fokus dan nyaman dalam belajar, karena ketika ada yang tidak kamu pahami bisa langsung ditanyakan.

PELAJARI DI EDUTECH ?


HTML, CSS & Bootstrap

Kamu akan memahami bagaimana HTML bekerja menggunakan library bootstrap sama seperti bagaimana web kami di buat.

Javascript & Jquery

Implementasi programming client side - front end yang akan membantu kamu membangun aplikasi berbasis web pada saat sudah memahami back end dalam PHP.

PHP Basic

Di PHP Basic, kamu akan mempelajari native PHP seperti syntax, variabels, data type, function, array hingga session dan juga studi kasus. Jadwal kamu dari hari senin sampai jumat adalah

  • 2 jam teori.
  • 3 jam tugas individu (didampingi expert)
  • 3 Jam tugas kelompok phase 2 (didampingi expert)
